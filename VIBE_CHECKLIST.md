# The Anti-Slop Vibe Checklist
- [ ] **Asymmetry Check**: Is it purposefully spaced or just "centered everything"?
- [ ] **Radix/Shadcn**: Are we using `<Dialog>` instead of a custom `div`?
- [ ] **States**: Do buttons have `:hover`, `:active`, and `:focus-visible`?
- [ ] **Motion**: Is there an `initial={{ opacity: 0, y: 20 }}` entrance?
- [ ] **The Generic Test**: Does this look like "Default AI"? If so, add a unique design token.
